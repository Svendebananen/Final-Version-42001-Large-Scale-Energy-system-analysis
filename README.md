## Parameter update structure

To keep a simple overview of Balmorel model changes, use:

- `/home/runner/work/42001-Large-Scale-Energy-system-analysis/42001-Large-Scale-Energy-system-analysis/Svendebananen/42001-Large-Scale-Energy-system-analysis/parameter_change_log.csv`

Add one row for every parameter-related change you make.

### Required columns

- `date`: Change date (YYYY-MM-DD)
- `author`: Who made the change
- `file`: File/path updated in the model setup
- `parameter`: Parameter name
- `old_value`: Previous value
- `new_value`: New value
- `reason`: Why it was changed
- `status`: `planned`, `implemented`, or `validated`
